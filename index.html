CSS Introducción

¿Qué es CSS?

  1. CSS significa hojas de estilo en cascada
  2. CSS describe cómo se deben mostrar los elementos HTML en la pantalla, el papel o en otros medios
  3. Las hojas de estilo externas se almacenan en archivos CSS
  4. CSS ahorra mucho trabajo. Puede controlar el diseño de varias páginas web a la vez

¿Por qué usar CSS?

CSS se usa para definir estilos para sus páginas web, incluido el diseño, el diseño y las variaciones en la visualización para diferentes dispositivos y tamaños de pantalla.

CSS Sintaxis

Una regla CSS consta de un selector y una declaración de bloque.

Sintaxis de CSS

  • El selector de puntos para el elemento HTML que desea estilo.
  • La declaración bloque contiene una o más declaraciones por separado punto y coma.
  • Cada declaración incluye una propiedad CSS nombre y un valor, separados por un signo de dos puntos.
  • Múltiples declaraciones CSS se separan con punto y coma, y la declaración los bloques están rodeados de llaves.

CSS Selectores

Los selectores de CSS

Los selectores de CSS se utiliza para "encontrar" (o seleccionar) los elementos HTML que desea que tengan estilos. Podemos dividir los selectores de CSS en cinco categorías:

  • Selectores simples(seleciona los elementos basado en el nombre,id,classe)
  • Combinador de selectores(seleciona elementos basado en una relación especifica entre ellos)
  • Pseudo-selectores de clase(seleciona elementos basados en un cierto estado)
  • Pseudo-elementos de elemento(selecciona y da estilos a una parte de un elemento)
  • Selectores de atributo(selecciona elementos con base en un atributo o valor)

Selector de elemento

El selector de elemento da estilos a los elementos de html basado en el nombre del elemento 

            
            p{ 
              text-align:center; 
              color:red; 
            }

Selector de id

El selector de id se utiliza el atributo id de un elemento HTML para seleccionar un elemento específico. El id de un elemento es único dentro de una página, por lo que el selector de id es se utiliza para seleccione un elemento único! Para seleccionar un elemento con un id específico, escriba un hash (#) de caracteres, seguido por el id del elemento. 

            
            
          #para1 {
            text-align:center;
            color:red;    
            }

El CSS Selector de clase

El selector de clase selecciona los elementos HTML con un determinado atributo de clase. Para seleccionar los elementos de una determinada clase, escribir un punto (.) carácter, seguido por el nombre de la clase. 

            
              .center {
                text-align:center;
                color: red;
                  }

También puede especificar que sólo determinados elementos HTML debe ser afectada por una clase. 

            
              p.center{
                text-align:center;
                color:red;
              }

Los elementos HTML también puede referirse a más de una clase.

            
            .center .large{
              color:red;
              border:2px solid red;
            }

El CSS Selector Universal

El CSS Selector Universal

            
              *{
                text-align:center;
                color:blue;
              }

Como agregar css

Cuando un navegador lee una hoja de estilo, formateará el documento HTML de acuerdo con la información de la hoja de estilo.

Tres formas de agregar Css

  • Css externo
  • Css interno
  • CSS en línea

CSS externo

Cada página HTML debe incluir una referencia al archivo de hoja de estilo externo dentro del elemento <link>, dentro de la sección de encabezado. 

  
   <!DOCTYPE   html>
    <html>
      <head>
        <link rel="stylesheet" href="mystyle.css">
    </head>
     <body>

      <h1>This is a heading</h1>
      <p>This is a paragraph.</p>

     </body>
    </html>

Una hoja de estilo externa se puede escribir en cualquier editor de texto y debe guardarse con una extensión .css. El archivo .css externo no debe contener ninguna etiqueta HTML. Así es como se ve el archivo "mystyle.css":

CSS interno

Se puede usar una hoja de estilo interna si una sola página HTML tiene un estilo único. El estilo interno se define dentro del elemento <style>, dentro de la sección de cabecera. 

  
   <!DOCTYPE   html>
    <html>
      <head>
        <link rel="stylesheet" href="mystyle.css">
      <style>
            p {
           background-color: linen;
            }

         h1 {
           color: maroon;
             margin-left: 40px;
            }
     <style>
    </head>
     <body>

     <h1>This is a heading </h1>
      <p>This is a paragraph.</p>

     </body>
    </html>

CSS en línea

Se puede usar un estilo en línea para aplicar un estilo único a un solo elemento. Para usar estilos en línea, agregue el atributo de estilo al elemento relevante. El atributo de estilo puede contener cualquier propiedad CSS. 

            
               <!DOCTYPE   html>
                 <html>
                 <body>
                <h1> style="color:blue;text-align:center;">This is a heading </h1>
                <p> style="color:red;">This is a paragraph. </p>
                  </body>
                   </html>

Orden en cascada

Orden en cascada

odos los estilos en una página "caerán en cascada" en una nueva hoja de estilo "virtual" según las siguientes reglas, donde el número uno tiene la prioridad más alta:

  1. Estilo en línea (dentro de un elemento HTML)
  2. Hojas de estilo externas e internas (en la sección de cabecera) predeterminado del navegador
  3. predeterminado del navegador

Por lo tanto, un estilo en línea tiene la prioridad más alta y anulará los estilos externos e internos y los valores predeterminados del navegador.

Las propiedades de margen de CSS se utilizan para crear espacio alrededor de los elementos, fuera de cualquier borde definido. Con CSS, tienes un control total sobre los márgenes. Hay propiedades para establecer el margen para cada lado de un elemento (arriba, derecha, abajo e izquierda).

CSS Margin Collapse

Top and bottom margins of elements are sometimes collapsed into a single margin that is equal to the largest of the two margins. This does not happen on left and right margins! Only top and bottom margins! Look at the following example:

              
                h1 {
                margin: 0 0 3.125rem 0;
                  }

              h2 {
              margin: 1.25rem 0 0 0;
              }

En el ejemplo anterior, el elemento <h1> tiene un margen inferior de 3.125rem y el elemento <h2> tiene un margen superior establecido en 1.25rem. El sentido común sugeriría que el margen vertical entre el <h1> y el <h2> sería un total de 4.375rem (3.125rem + 1.25rem). Pero debido a la combinación de márgenes, el margen real resulta ser de 3.125rem.

  • length: especifica un padding en px, pt, cm, etc.
  • %: especifica un padding en % del ancho del elemento contenedor
  • inherit: especifica que el padding se herede del elemento padre

Note: Negative values are not allowed.

  
            div {
            padding-top: 3.125rem;
            padding-right: 1.875rem;
            padding-bottom: 3.125rem;
            padding-left: 5rem;
          }

Padding - Shorthand Property

To shorten the code, it is possible to specify all the padding properties in one property. The padding property is a shorthand property for the following individual padding properties:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Entonces, aquí es cómo funciona:

Si la propiedad de padding tiene cuatro valores:padding: 1.5625rem 3.125rem 4.6875rem 6.25rem;

  • top padding is 1.5625rem
  • right padding is 3.125rem
  • bottom padding is 4.6875rem
  • left padding is 6.25rem
  
                div {
                padding: 1.5625rem 3.125rem 4.6875rem 6.25rem;
                }

If the padding property has three values:padding: 1.5625rem 3.125rem 4.6875rem;

  
              div {
              padding: 1.5625rem 3.125rem 4.6875rem;
              }

f the padding property has two values:padding: 1.5625rem 3.125rem;

  • Paddings superiores e inferiores son 1.5625rem
  • Paddings derechos e izquierdos son 3.125rem

  
    div {
  padding: 1.5625rem 3.125rem;
}

If the padding property has one value:padding: 1.5625rem;

  • all four paddings are 1.5625rem

          div {
          padding: 1.5625rem;
        }

Padding y elementos Width

The CSS width property specifies the width of the element's content area. The content area is the portion inside the padding, border, and margin of an element (the box model). So, if an element has a specified width, the padding added to that element will be added to the total width of the element. This is often an undesirable result.

  
                div {
              width: 18.75rem;
              padding: 1.5625rem;
            }

Para mantener el ancho en 18.75rem, sin importar la cantidad de padding, puedes utilizar la propiedad box-sizing. Esto hace que el elemento mantenga su ancho real; si aumentas el padding, el espacio de contenido disponible se reducirá.

  
                    div {
                  width: 18.75rem;
                  padding: 1.5625rem;
                  box-sizing: border-box;
                }

Las propiedades de padding en CSS se utilizan para generar espacio alrededor del contenido de un elemento, dentro de cualquier borde definido. Con CSS, tienes control total sobre el padding. Hay propiedades para establecer el padding en cada lado de un elemento (superior, derecho, inferior e izquierdo).

Padding - Individual Sides

Las propiedades de height y width se utilizan para establecer la altura y el ancho de un elemento. Las propiedades de height y width no incluyen el padding, los bordes ni los márgenes. Establecen la altura/ancho del área dentro del padding, borde y margen del elemento.

CSS height and width Values

Las propiedades de altura (height) y ancho (width) pueden tener los siguientes valores:

  • auto: Esto es el valor por defecto. El navegador calcula la altura y el ancho.
  • length: Define la altura/ancho en px, cm, etc.
  • %: Define la altura/ancho en porcentaje del bloque contenedor.
  • initial: Establece la altura/ancho a su valor por defecto.
  • inherit: La altura/ancho se heredará del valor del elemento padre.

  • La propiedad max-width se utiliza para establecer el ancho máximo de un elemento.

CSS Box Model

En CSS, el término "modelo de caja" se utiliza cuando se habla de diseño y maquetación. El modelo de caja CSS es básicamente una caja que envuelve a cada elemento HTML. Está compuesto por: márgenes, bordes, relleno y el contenido real. La imagen a continuación ilustra el modelo de caja: [imagen del modelo de caja]

Explicación de las diferentes parte

  • Contenido: El contenido de la caja, donde aparecen texto e imágenes
  • Padding: Limpia un área alrededor del contenido. El padding es transparente
  • Borde: Un borde que rodea el padding y el contenido
  • Margen: Limpia un área fuera del borde. El margen es transparente

El modelo de caja nos permite agregar un borde alrededor de los elementos y definir el espacio entre ellos.
